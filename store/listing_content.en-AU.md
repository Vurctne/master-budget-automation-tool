# Store Listing Draft

## Product name
Master Budget Automation Tool

## Short description
Import Compass Expense Sub-Program data into a Master Budget workbook without OFFSET/MATCH formulas.

## Description
Master Budget Automation Tool helps school finance staff import Expense Sub-Program exports from Compass into a Master Budget workbook quickly and consistently.

The app reads a local Compass export in CSV, XLSX, or XLSM format, maps the values into the Master sheet, and saves a new output workbook so the original template stays unchanged.

Key benefits:
- Imports values directly into the Master sheet
- Refreshes the Compass sheet when present
- Preserves macro button bindings on Windows when Microsoft Excel is installed
- Keeps blank source values blank
- Highlights mismatch items for review
- Inserts source-only account codes and sub-program codes in numeric order
- Works with local files and does not require sign-in

## What's new in this version?
- Improved file validation to prevent overwriting the source workbook or stripping macros by mistake
- Added duplicate source-code detection for safer imports
- Improved CSV encoding compatibility for Windows exports
- Added Windows version metadata and Store-ready installer packaging
