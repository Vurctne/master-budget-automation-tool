Windows EXE note
================

This source package is prepared for a Windows EXE build.
Run build_windows_exe.bat on a Windows PC with Python installed.
That will create:

dist\Master Budget Automation Tool v1.0.2.exe

The Compass download screenshots used by the in-app Instructions window are included in the assets folder.

Suggestion note shown in the app:
Please send suggestions to ivan.wang@education.vic.gov.au

For Microsoft Store EXE submission preparation, use:
build_store_installer.ps1
